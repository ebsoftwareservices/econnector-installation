## Install

Run `install.bat` to install the econnector, it will install all the needed files into `C:\Econnector` directory.


## Uninstall 

Run `uninstall.bat` to uninstall the econnector, it couldn't remove the `C:\Econnector` directory, you have to maunally clean it.

## Debug

Check the logs in `C:\Econnector\logs`.
